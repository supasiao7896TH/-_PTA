# -_PTA

คลังความรู้ (knowledge base) ส่วนตัวสำหรับเก็บรวบรวมข้อมูล บันทึก และเอกสารที่เกี่ยวข้องกับ PTA

## การใช้งาน

เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์โดยตรง (ดับเบิลคลิก หรือ File > Open) ไม่ต้องมีอินเทอร์เน็ตหรือ server ใดๆ
- แท็บ **ค้นหา**: พิมพ์คำค้นหา (ไทยหรืออังกฤษ) เพื่อค้นข้อความจากเอกสารทั้งหมด พร้อมอ้างอิงชื่อไฟล์และหมายเลขหน้า
- แท็บ **ห้องสมุดเอกสาร**: ดูรายชื่อเอกสารทั้งหมดพร้อมคำอธิบาย
- ปุ่ม "เปิด PDF" / "ดาวน์โหลด" ในแต่ละผลลัพธ์ใช้เปิด/บันทึกไฟล์ PDF ต้นฉบับ (ฝังอยู่ในไฟล์ `index.html` แล้ว)

## การ build ใหม่

ถ้ามีการเพิ่ม/แก้ไข PDF หรือแก้ไข UI ใน `build/template.html` ให้ build ไฟล์ `index.html` ใหม่:

```
pip install -r build/requirements.txt
python build/build_kb.py
```

สคริปต์จะดึงข้อความจาก PDF ทุกไฟล์ในโฟลเดอร์นี้ เข้ารหัส base64 แล้วรวมเข้ากับ `build/template.html` เพื่อสร้าง `index.html` ใหม่ทั้งไฟล์
